# 📖 [Metrics Web App ] <a name="about-project"></a>

**Metrics Web App** is a mobile app developed with react and redux, it helps users to access countries
and cities and retrieve air quality details of each city. The air quality data is fetched from an APi.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites <a name="prerequisites">

In order to run this project you need:

- Visual Studio Code
- Node JS

### Setup <a name="setup">

Clone this repository to your desired folder:

Example commands:

```sh
  mkdir metrics-app
  cd metrics-app
  git clone git@github.com:demesameneshoa/metrics-web-app.git
```

### Insatll <a name="install">

Initiate npm by running the following command

```
npm install
npm start
```
